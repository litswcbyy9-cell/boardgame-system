\section{项目数据库脚本清单}

本项目包含以下数据库脚本，均位于 \code{db/} 目录下：

\subsection{初始化脚本}

\begin{enumerate}
  \item \code{bootstrap.sql}：创建数据库和初始用户
  \item \code{create-app-user.sql}：创建应用用户和权限
  \item \code{init/01\_schema.sql}：创建所有表、约束、索引、触发器
  \item \code{init/02\_procedures.sql}：创建存储过程
  \item \code{init/03\_seed.sql}：插入基础种子数据
  \item \code{init/04\_views.sql}：创建视图
  \item \code{init/05\_bulk\_seed.sql}：插入大量测试数据（可选）
  \item \code{init/06\_security\_grants.sql}：配置权限
\end{enumerate}

\subsection{脚本内容概览}

\subsubsection{01\_schema.sql}

该脚本创建以下对象：

\begin{itemize}
  \item 11 个数据表，覆盖门店、后台账号、登录会话、游戏目录、桌位状态、会员资料、预约、对局和战绩等核心对象
  \item 主键和外键约束
  \item CHECK 约束和 UNIQUE 约束
  \item 复合索引和单列索引
  \item 4 个触发器
\end{itemize}

\subsubsection{02\_procedures.sql}

该脚本创建以下存储过程：

\begin{itemize}
  \item \code{sp\_reserve\_table}：创建预约
  \item \code{sp\_checkin\_start\_session}：预约入场
  \item \code{sp\_start\_walkin\_session}：现场开台
  \item \code{sp\_end\_session\_settle}：结算关台
  \item \code{sp\_insert\_game\_record}：录入战绩
  \item \code{sp\_cancel\_reservation}：取消预约
  \item \code{sp\_report\_daily\_revenue}：日收入统计
  \item \code{sp\_report\_game\_popularity}：游戏热度排行
  \item \code{sp\_report\_table\_utilization}：桌位利用率分析
  \item \code{sp\_recommend\_games}：智能桌游推荐
  \item \code{sp\_recommend\_tables}：智能桌位调度
\end{itemize}

\subsubsection{03\_seed.sql}

该脚本插入以下基础数据：

\begin{itemize}
  \item 1 个门店（示例门店）
  \item 25 个桌游（含类型、难度、平均时长和推荐权重）
  \item 12 个桌位（3x4 网格）
  \item 30 个示例会员
\end{itemize}

\subsubsection{04\_views.sql}

该脚本创建以下视图：

\begin{itemize}
  \item \code{v\_table\_status\_floor}：平面图查询
  \item \code{v\_leaderboard}：排行榜
  \item \code{v\_session\_billing\_detail}：账单明细
  \item \code{v\_game\_catalog\_with\_stats}：游戏热度
  \item \code{v\_game\_recommendation\_features}：推荐特征聚合
\end{itemize}

\subsubsection{05\_bulk\_seed.sql}

该脚本插入大量测试数据（可选）：

\begin{itemize}
  \item 约 66 个会员
  \item 约 360 条历史对局记录
  \item 约 360 条战绩记录
\end{itemize}

用于性能测试和压测。

\subsubsection{06\_security\_grants.sql}

该脚本配置以下权限：

\begin{itemize}
  \item \code{bg\_readonly}：只读账号，用于报表查询
  \item \code{bg\_app}：应用账号，具有 INSERT、UPDATE、DELETE 权限
\end{itemize}
